# Idea
 
I think an AI streamer for coffee chat might be welcome?  本质上这类产品是我从metastable时候就一直想做的，C2C的agent, user generated agent as a service，价值来自于私域信息而不是缺乏稀缺性的AI基础模型，信息和机会掌握在少数人手里，但是少数人的时间有限，通过AI将人有限的精力放大，创造AI分身代替本体直播。
这件事本身也能帮助我低时间成本地解决社交需求，通过一个24小时直播的agent来寻找潜在的伴侣。

这个模式要比卖AI agent作为产品好一些，毕竟成本均摊到所有观众，而且有天然的流量入口，发掘到强烈付费意愿的用户再进一步卖给他私人agent服务。
这个想法比deepmatch也受众更广，deepmatch的设计是用来找人的搜索引擎，通过AI来实现模糊的筛选和隐私（AI看照片来筛选符合喜好的人，从而照片不被人看到），受众是目的性明确，知道自己想要什么的人，直接的细分市场是性交易和包养。我们必须承认，世界上存在大量廉价的注意力，不是所有人都知道自己想要什么。
