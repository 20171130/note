# Log

A daily log is a chronological record of events we experience throughout the day, consisting of objective information (fact / hypothesis / experience) interleaved with our free thoughts and prose.

## Steps

1. Read today's `log/YYYY-MM-DD.md` (Henry's timezone from [user.mdc](../../rules/user.mdc)). If today's file is empty or missing, look at yesterday's to find the last entry.
2. Append experience in this session into today's daily log.
3. Notify Henry of the entry added.

Add context: supplement Henry's notes with my own observations from what we experienced together. Web search when information is missing. Ask Henry when something is still unclear.
